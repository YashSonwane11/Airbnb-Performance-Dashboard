# 🏠 Airbnb Performance Dashboard

## 📊 Project Overview
This project analyzes Airbnb data (279K+ listings and 5M+ reviews) to uncover key insights about market trends, customer behavior, and city-level performance using Power BI.

---

## 🚀 Key Features
- 📍 Market share analysis across 10 global cities
- 📈 Trend analysis using Pareto and seasonality insights
- ⭐ Ratings analysis (cleanliness, value, accuracy)
- 🔍 Trust Metric to evaluate host authenticity
- 📊 Interactive drill-through dashboards

---

## 🛠️ Tech Stack
- Power BI
- DAX
- Power Query
- ETL

---

## 📂 Dataset
- Listings: 279,000+
- Reviews: 5.3 Million+
- Cities: 10
- Hosts: 182,000+

---

## 📌 Key Insights
- Paris, NYC, and Sydney contribute ~48% of total engagement
- Most users leave reviews only once (98% ≤ 3 reviews)
- Cleanliness and value-for-money are lowest-rated metrics
- Seasonal trends peak during summer and holiday periods
